# utxt README

Syntax highlighting for text files.




Add below code to ~/.vscode/extensions/extensions.json

```
    {
        "identifier": {
            "id": "alxnull.utxt"
        },
        "version": "0.0.1",
        "location": {
            "$mid": 1,
            "path": "/home/udo/.vscode/extensions/utxt",
            "scheme": "file"
        },
        "relativeLocation": "utxt"
    },
```
